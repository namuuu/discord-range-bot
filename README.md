# Range Bot, a discord coaching utility bot

## Lancer l'application
Afin de pouvoir lancer le bot correctement, il est indispensable de créer un fichier `.env` avec le contenu suivant :
```js
DISCORD_BOT_TOKEN=...
CLIENT_ID=...
DISCORD_GUILD_ID=...
FIREBASE_PROJECT_ID=...
FIREBASE_CLIENT_EMAIL=...
FIREBASE_PRIVATE_KEY="..."
```
